FFmpeg: Essential Commands for Video and Audio Processing

January 5, 2026 | Tools

FFmpeg is the Swiss Army knife of multimedia processing. It's a command-line tool that can convert, compress, stream, and manipulate virtually any audio or video format. While the documentation can be overwhelming, most day-to-day tasks require just a handful of commands.

Installing FFmpeg

FFmpeg is available on all major platforms: 

# Ubuntu/Debian
sudo apt install ffmpeg

# macOS (with Homebrew)
brew install ffmpeg

# Arch Linux
sudo pacman -S ffmpeg

# Windows (with Chocolatey)
choco install ffmpeg

Basic Syntax

FFmpeg follows a consistent pattern: input options, input file, output options, output file. 

ffmpeg [input options] -i input.mp4 [output options] output.mp4

The -i flag specifies input files. Everything before -i applies to the input, everything after applies to the output.

Convert Between Formats

The simplest and most common use case. FFmpeg automatically detects formats from file extensions:

Video format conversion:

# MKV to MP4
ffmpeg -i video.mkv video.mp4

# AVI to MP4
ffmpeg -i video.avi video.mp4

# MOV to MP4 (common for iPhone videos)
ffmpeg -i video.mov video.mp4

# WebM to MP4
ffmpeg -i video.webm video.mp4

Audio format conversion:

# WAV to MP3
ffmpeg -i audio.wav audio.mp3

# FLAC to MP3 (with quality setting)
ffmpeg -i audio.flac -b:a 320k audio.mp3

# MP3 to AAC
ffmpeg -i audio.mp3 -c:a aac audio.m4a

# OGG to MP3
ffmpeg -i audio.ogg audio.mp3

Compress Video Files

Reducing file size is one of FFmpeg's most valuable capabilities. The CRF (Constant Rate Factor) controls quality vs size tradeoff. 

# Standard compression with H.264 (CRF 23 is default, lower = better quality)
ffmpeg -i input.mp4 -c:v libx264 -crf 23 output.mp4

# Higher compression (smaller file, slightly lower quality)
ffmpeg -i input.mp4 -c:v libx264 -crf 28 output.mp4

# Better compression with H.265/HEVC (slower but smaller files)
ffmpeg -i input.mp4 -c:v libx265 -crf 28 output.mp4

# Compress and also reduce resolution
ffmpeg -i input.mp4 -vf "scale=1280:720" -c:v libx264 -crf 23 output.mp4

CRF Guidelines: For H.264, use 18-28. CRF 18 is visually lossless, 23 is default, 28 is noticeable but acceptable. For H.265, add about 4-6 to these values (CRF 28 in H.265 â‰ˆ CRF 23 in H.264).

Extract Audio from Video

Pull the audio track out of a video file: 

# Extract as MP3
ffmpeg -i video.mp4 -vn -b:a 320k audio.mp3

# Extract original audio without re-encoding (faster, no quality loss)
ffmpeg -i video.mp4 -vn -c:a copy audio.aac

# Extract as high-quality WAV
ffmpeg -i video.mp4 -vn audio.wav

The -vn flag means "no video" in the output.

Trim and Cut Videos

Extract a portion of a video by specifying start time and duration: 

# Cut from 00:01:30 for 45 seconds
ffmpeg -i input.mp4 -ss 00:01:30 -t 00:00:45 -c copy output.mp4

# Cut from 00:01:30 to 00:02:15 (using -to instead of -t)
ffmpeg -i input.mp4 -ss 00:01:30 -to 00:02:15 -c copy output.mp4

# Cut first 30 seconds
ffmpeg -i input.mp4 -t 30 -c copy output.mp4

# Remove first 10 seconds
ffmpeg -i input.mp4 -ss 10 -c copy output.mp4

Tip: Using -c copy is much faster because it doesn't re-encode, but cuts may not be frame-accurate. For precise cuts, remove -c copy to re-encode.

Resize and Scale Videos

Change video resolution: 

# Scale to specific resolution
ffmpeg -i input.mp4 -vf "scale=1920:1080" output.mp4

# Scale width to 1280, calculate height automatically (maintain aspect ratio)
ffmpeg -i input.mp4 -vf "scale=1280:-1" output.mp4

# Scale height to 720, calculate width automatically
ffmpeg -i input.mp4 -vf "scale=-1:720" output.mp4

# Scale to 50% of original size
ffmpeg -i input.mp4 -vf "scale=iw/2:ih/2" output.mp4

Create GIFs from Video

Convert video clips to animated GIFs: 

# Simple GIF (lower quality, larger file)
ffmpeg -i input.mp4 -vf "fps=10,scale=480:-1" output.gif

# High-quality GIF using palette (two-pass)
ffmpeg -i input.mp4 -vf "fps=15,scale=480:-1:flags=lanczos,palettegen" palette.png
ffmpeg -i input.mp4 -i palette.png -lavfi "fps=15,scale=480:-1:flags=lanczos[x];[x][1:v]paletteuse" output.gif

# GIF from specific portion of video
ffmpeg -ss 00:00:05 -t 3 -i input.mp4 -vf "fps=10,scale=320:-1" output.gif

Extract Frames as Images

Pull individual frames or create thumbnails: 

# Extract single frame at specific timestamp
ffmpeg -i input.mp4 -ss 00:00:10 -frames:v 1 thumbnail.png

# Extract one frame per second
ffmpeg -i input.mp4 -vf "fps=1" frame_%04d.png

# Extract all frames (warning: creates many files!)
ffmpeg -i input.mp4 frame_%04d.png

# Extract frame every 10 seconds
ffmpeg -i input.mp4 -vf "fps=1/10" frame_%04d.png

Create Video from Images

Turn a sequence of images into a video (useful for timelapses): 

# Create video from numbered images (img001.png, img002.png, etc.)
ffmpeg -framerate 30 -i img%03d.png -c:v libx264 -pix_fmt yuv420p output.mp4

# Create video with specific frame rate and quality
ffmpeg -framerate 24 -i frame_%04d.png -c:v libx264 -crf 18 -pix_fmt yuv420p timelapse.mp4

# Use glob pattern for arbitrary filenames
ffmpeg -framerate 30 -pattern_type glob -i '*.png' -c:v libx264 -pix_fmt yuv420p output.mp4

Merge/Concatenate Videos

Combine multiple video files into one: 

# First, create a text file listing the videos:
# files.txt:
# file 'video1.mp4'
# file 'video2.mp4'
# file 'video3.mp4'

# Then concatenate (same codecs, no re-encoding)
ffmpeg -f concat -safe 0 -i files.txt -c copy output.mp4

# Concatenate with re-encoding (different codecs/resolutions)
ffmpeg -f concat -safe 0 -i files.txt -c:v libx264 -c:a aac output.mp4

Add Audio to Video

Replace or add an audio track: 

# Replace audio track
ffmpeg -i video.mp4 -i audio.mp3 -c:v copy -c:a aac -map 0:v:0 -map 1:a:0 output.mp4

# Add audio to silent video
ffmpeg -i video.mp4 -i music.mp3 -c:v copy -c:a aac -shortest output.mp4

# Remove audio entirely
ffmpeg -i input.mp4 -an -c:v copy output.mp4

Change Video Speed

Speed up or slow down video playback: 

# Speed up 2x (video and audio)
ffmpeg -i input.mp4 -filter_complex "[0:v]setpts=0.5*PTS[v];[0:a]atempo=2.0[a]" -map "[v]" -map "[a]" output.mp4

# Slow down to 0.5x speed
ffmpeg -i input.mp4 -filter_complex "[0:v]setpts=2.0*PTS[v];[0:a]atempo=0.5[a]" -map "[v]" -map "[a]" output.mp4

# Speed up video only (no audio)
ffmpeg -i input.mp4 -vf "setpts=0.5*PTS" -an output.mp4

Rotate and Flip Videos

Fix orientation or create mirror effects: 

# Rotate 90 degrees clockwise
ffmpeg -i input.mp4 -vf "transpose=1" output.mp4

# Rotate 90 degrees counter-clockwise
ffmpeg -i input.mp4 -vf "transpose=2" output.mp4

# Rotate 180 degrees
ffmpeg -i input.mp4 -vf "transpose=1,transpose=1" output.mp4

# Flip horizontally (mirror)
ffmpeg -i input.mp4 -vf "hflip" output.mp4

# Flip vertically
ffmpeg -i input.mp4 -vf "vflip" output.mp4

Get Video Information

Inspect media file properties: 

# Basic info
ffmpeg -i input.mp4

# Detailed info with ffprobe (comes with FFmpeg)
ffprobe input.mp4

# JSON output for scripting
ffprobe -v quiet -print_format json -show_format -show_streams input.mp4

# Get just the duration
ffprobe -v error -show_entries format=duration -of default=noprint_wrappers=1:nokey=1 input.mp4

Useful Flags Reference

Quick reference for commonly used FFmpeg flags:

-i Input file
-c:v Video codec (libx264, libx265, copy)
-c:a Audio codec (aac, mp3, copy)
-c copy Copy streams without re-encoding (fast)
-vf Video filter (scale, rotate, crop, etc.)
-af Audio filter
-ss Start time (seek position)
-t Duration
-to End time
-vn No video (audio only output)
-an No audio (video only output)
-crf Quality (0-51, lower is better)
-b:v Video bitrate
-b:a Audio bitrate
-y Overwrite output without asking

Conclusion

FFmpeg is incredibly powerful, and these commands cover the vast majority of everyday use cases. The key to using FFmpeg effectively is understanding the basic syntax pattern and knowing which flags control video (-c:v, -vf) versus audio (-c:a, -af).

When in doubt, FFmpeg's documentation is extensive, and the ffmpeg -h command provides quick help. For more complex filters and operations, the FFmpeg Filters Documentation is an invaluable resource.
